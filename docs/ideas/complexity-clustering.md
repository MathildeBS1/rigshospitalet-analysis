# Complexity Clustering (K-Means)

Cluster operations into natural complexity tiers using K-means.

**Features:** staff count, equipment count, actual prep time, specialty, overtime minutes

**Goal:** Find K clusters (e.g. light / medium / heavy), then show that planned prep time doesn't vary across clusters the way actual prep time does — making Mathilde's correlation finding more granular and visual.

**Output:** A complexity label per operation type that could inform scheduling directly.
